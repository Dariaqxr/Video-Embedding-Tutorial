How to Embed a YouTube Video on Your Website

Follow this simple, step-by-step guide with clear screenshots and code to successfully add videos to your site.

  • Find Video
  • Share Button
  • Select Embed
  • Copy Code
  • Paste HTML
  • Example Video

Step 1: Find Your Desired YouTube Video

Go to YouTube and find the video you want to embed. Choose a video that is publicly available and appropriate for embedding.

Step 2: Click the "Share" Button

Below the video player, click the **Share** button. A popup will appear with sharing options.

Step 3: Select "Embed"

In the popup, click on the **Embed** option. This will open a window with HTML embed code.

Step 4: Copy the Embed Code

Click the **Copy** button next to the code. This code can now be pasted into your website's HTML.

Step 5: Paste Code into Your HTML File

Open your HTML file in a text editor (e.g., VS Code or Notepad++) and paste the code where you want the video to appear.

<iframe width="560" height="315" src="https://www.youtube.com/embed/YOUR_VIDEO_ID" 
frameborder="0" allowfullscreen></iframe>

Please replace `YOUR_VIDEO_ID` in the `src` attribute with the actual ID of your YouTube video.
